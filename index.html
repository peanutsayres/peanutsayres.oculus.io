What This Is

Accessus Publica provides temporary, paid access to professional-grade tools that are otherwise difficult to own, store, or justify full-time.

We start with one asset. One geography. One question.

This is not a marketplace.
This is not a studio.
This is not instruction or community programming.

This is access.

Who This Is For

This pilot is intentionally narrow. We are currently speaking with:

  • Weaving & fiber guilds
  • Universities and colleges
  • Institutional studios

If you need temporary access to a professional loom for real work—this may fit.

Phase 0 Pilot

Objective

To determine whether professional loom access generates repeat, paid utilization without owning a facility, vehicle fleet, or permanent inventory.

If it does, we proceed carefully. If it does not, we stop.

No sunk-cost continuation.

The Loom

Professional floor loom (Macomber / equivalent)

  • Maintained and functional
  • Cleaned, inspected, insured
  • Not pristine
  • Not upgraded
  • Not bundled with supplies

This pilot tests access, not perfection.

What We're Learning

This pilot helps us understand what works: pricing, delivery preferences, lease duration, and whether this model serves real demand.

We are measuring:

  • Price acceptance
  • Optimal lease duration (30 days? 90 days? Semester-length?)
  • Extension behavior (do lessees renew?)
  • Delivery vs pickup preference
  • Institutional vs guild demand
  • Logistics friction

Revenue matters. Learning matters more.

What Happens After Phase 0

If the pilot succeeds:

  • Add 1–2 additional looms or adjacent fiber assets
  • Formalize referral pathways
  • Begin Phase 1 facility modeling (carefully)

If results are mixed:

  • Adjust pricing, radius, or terms once
  • Extend evaluation period
  • Re-evaluate

If demand doesn't materialize:

  • We exit cleanly and document findings
  • Loom is sold or repurposed

Stopping cleanly is a strength, not a failure.

About Accessus Publica

Why This Exists

Accessus Publica exists to test whether temporary access to professional tools serves demand that ownership and studios don't.

Phase 0 is a single-asset pilot to gather signal. If it works, we expand. If it doesn't, we document findings and move on.

What We Believe

  • Ownership is often inefficient
  • Access can be dignified
  • Temporary infrastructure can be sufficient
  • Stopping cleanly is a strength

What This Is Not

  • Not a subscription service
  • Not a community program
  • Not a permanent facility

The Principle
